# DT Doctor

DT Doctor is a static analysis tool that helps diagnose Devicetree-related build errors.
To display the application DT Doctor, right-click on the application > Static Code Analysis > DT Doctor.

![DT Doctor menu](/img/zw/static-code-analysis/dt-doctor/dt-doctor.png)

The report is displayed in the Terminal. It provides detailed information about the possible cause of the error and how to fix it.

![DT Doctor terminal](/img/zw/static-code-analysis/dt-doctor/dt-doctor-terminal.png)
